Caso donde tengamos acceso a un cliente git en el servidor 

1. Accedemos al servidor mediante SSH
2. Nos movemos al directorio web (/web)
3. Crearemos un directorio nuevo para el proyecto (mkdir alimentos) y nos redirigiremos a él (cd alimentos)
4. En caso de tener acceso a un cliente git clonaríamos el repositorio con el comando git clone, pero como no es el caso descargaremos 
	el repositorio con el comando wget --no-check-certificate https://github.com/Franciscojpf00/GRUPO4DAW20/archive/master.zip 
5. Descomprimimos el archivo descargado con el comando unzip master.zip
6. Para mayor comodida para acceder al proyecto, renombraremos la carpeta con el comando mv GRUPO4DAW20-master/ master
7. Ya estaría el proyecto desplegado, podremos acceder mediante la url: http://www.alessrodrgz.com.mialias.net/alimentos/master/web/
